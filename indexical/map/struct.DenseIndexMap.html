Struct DenseIndexMap 

Source 
pub struct DenseIndexMap<'a, K: IndexedValue + 'a, V, P: PointerFamily<'a>>(/* private fields */);
Expand description

A mapping from indexed keys to values, implemented densely with a vector.

This is more time-efficient than the SparseIndexMap for lookup, but it consumes more memory for missing elements.
