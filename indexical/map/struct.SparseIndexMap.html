pub struct SparseIndexMap<'a, K: IndexedValue + 'a, V, P: PointerFamily<'a>> { /* private fields */ }
Expand description

A mapping from indexed keys to values, implemented sparsely with a hash map.

This is more memory-efficient than the DenseIndexMap with a small number of keys.
