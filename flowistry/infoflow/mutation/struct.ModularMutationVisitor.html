pub struct ModularMutationVisitor<'a, 'tcx, F>where
    F: FnMut(Mutation<'_, 'tcx>),{ /* private fields */ }
Expand description

MIR visitor that invokes a callback for every Mutation in the visited object.

Construct the visitor with ModularMutationVisitor::new, then call one of the MIR Visitor methods.
